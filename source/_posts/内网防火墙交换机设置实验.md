命令配置步骤

创建VLAN 100，并将GigabitEthernet 1/0/1加入VLAN 100。

```
system-view
vlan 100
port gigabitethernet 1/0/1
quit` 


```

创建Vlan-interface 100，并配置其IP地址为192.168.1.1/24。

```
interface Vlan-interface 100
ip address 192.168.100.2 24
quit 

```
